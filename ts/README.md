# 简介

这是一个用 Javascript 写的调用 [Bilibili](https://www.bilibili.com) 各种 API 的库，
这个库是基于 `python` 版的一个 `clone`, API 正在慢慢完善，欢迎贡献！。

**注意：本仓库全都是异步操作，而且是 `ajax`，就是那种代码跑完了结果你的结果才刚刚出来的那种，不 `await` 后果自负。**

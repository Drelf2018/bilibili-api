## Python

## Javascript / Typescript

- 添加了注释！！！
- **传参方式改回了一个一个传参了，已废弃字典传参。** 主要是因为这样子注释写着没啥子用，而且写代码的时候 intellicode 效果差。

``` typescript
// 以前
get_info({
    aid: 2
});
// 现在
get_info(null, 2); // get_info(bvid, aid)，不传 bvid
```
